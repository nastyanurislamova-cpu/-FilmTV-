# Дашборд для анализа финансовых показателей

Интерактивный дашборд для анализа ключевых финансовых показателей компании на основе квартальной бухгалтерской отчетности.

## 📊 Обзор

Приложение предоставляет комплексный анализ финансовых показателей, включая:
- Динамику выручки и прибыли
- Показатели рентабельности (ROA, ROE, маржа прибыли)
- Показатели ликвидности (коэффициент текущей ликвидности, быстрой ликвидности)
- Показатели финансового рычага
- Трендовый анализ и прогнозирование

## 🚀 Возможности

- **Интерактивные визуализации**: Динамические графики квартальных трендов
- **Сравнительный анализ**: Сопоставление показателей по периодам
- **Финансовые коэффициенты**: Автоматический расчет важных финансовых индикаторов
- **Экспорт данных**: Скачивание результатов анализа в различных форматах
- **Адаптивный дизайн**: Работает на десктопе и мобильных устройствах

## 🛠️ Технологический стек

- **Python 3.11+**
- **Streamlit** - Фреймворк для веб-приложений
- **Pandas** - Обработка и анализ данных
- **Plotly** - Интерактивные визуализации
- **Docker** - Контейнеризация

## 📦 Установка

### Использование Docker (Рекомендуется)

```bash
docker build -t financial-analysis .
docker run -p 8501:8501 financial-analysis
```

### Локальная установка

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📈 Использование

1. Запустите приложение
2. Загрузите финансовые данные (формат Excel/CSV) или используйте сгенерированные примеры
3. Выберите метрики для анализа
4. Изучайте интерактивные графики и аналитику

## 📄 Формат данных

Приложение ожидает квартальные финансовые данные со следующей структурой:
- Квартал/Период
- Выручка
- Операционные расходы
- Чистая прибыль
- Совокупные активы
- Оборотные активы
- Краткосрочные обязательства
- Собственный капитал
